# Deno REST API

[Deno](https://deno.land/) is a simple, modern and secure runtime for JavaScript and TypeScript that uses V8 and is built in Rust.

### **Run a program**

    deno run --allow-net server.ts

| Methods  | Paths |
| --- | --- |
| GET  | /api/v1/items/  |
| GET  | /api/v1/items/:id  |
| POST  | /api/v1/items/  |
| PUT  | /api/v1/items/:id  |
| DELETE  | /api/v1/items/:id  |
